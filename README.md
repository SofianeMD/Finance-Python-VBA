## 1. Analyse de Corrélation (Python)
Ce script analyse la corrélation glissante entre le cours de l'action **Air France (AF.PA)** et le prix du **Pétrole Brent**. 
**Objectif** : Identifier les déconnexions fondamentales entre le coût du barils de pétrole et le cours de la compagnie aérienne.
**Librairies** : yfinance, pandas, matplotlib.

## 2. Automatisation Excel (VBA)
