# 1. Analyse de Corrélation (Python)
Ce script analyse la corrélation glissante entre le cours de l'action **Air France (AF.PA)** et le prix du **Pétrole Brent**. 
**Objectif** : Identifier les déconnexions fondamentales entre le coût du barils de pétrole et le cours de la compagnie aérienne, à des dates précises puis y chercher les raisons qui sont en générale liées à des annonces et/ou des informations, qu'elles soit macroéconomique ou liées à l'entreprise voir le secteur entier, à mon sens ce genre de code peut être utile en gestion de portfeuille, et en gestion de risque principalement.

**Librairies** : yfinance, pandas, matplotlib.
